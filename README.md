# SymfonyScheduler Module
[![Latest Stable Version](https://poser.pugx.org/spryker/symfony-scheduler/v/stable.svg)](https://packagist.org/packages/spryker/symfony-scheduler)

[![Minimum PHP Version](https://img.shields.io/badge/php-%3E%3D%208.3-8892BF.svg)](https://php.net/)

The SymfonyScheduler module provides integration with Symfony Scheduler component for scheduling and executing tasks in Spryker applications.

## Installation

```bash
composer require spryker/symfony-scheduler
```
